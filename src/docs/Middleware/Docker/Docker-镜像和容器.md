## 镜像和容器
### 镜像
- 把软件和软件所需要的运行环境打包到镜像文件中，通过镜像文件，能创建出对应的容器，就有了对应的软件。便于在linux中配置软件，更为方便

### 镜像和容器的关系
- 类似于类和对象的关系，通过类能创建很多对象，容器是通过静态的镜像文件所创建出来的动态实例
 - 容器可以被创建、启动、停止、删除、暂停等
